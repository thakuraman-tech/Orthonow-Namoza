# Namoza / OrthoNow — Developer & MarTech Architect Assignment

This repository contains the deliverables for the 3-part developer assignment for Namoza (agency) and OrthoNow (client, 9 orthopaedic clinics in Bengaluru).

## Repository Structure

* **[Task 01: GTM Event Schema](task1-gtm-schema.md)** — Comprehensive event measurement schema table, `dataLayer.push` snippets for the 3-step booking funnel, GA4 funnel exploration guide, and Google Ads optimization strategy.
* **[Task 02: Landing Page](task2-landing-page/index.html)** — Single-file, high-performance, mobile-first responsive landing page with inline CSS/JS and inline SVGs. Contains dynamic user tracking, zero-CLS input validation, and a custom thank-you state transition.
* **[Task 03: Integration Design](task3-integration-design.md)** — Integration architecture details mapping webhook capture to HubSpot CRM (using phone deduplication), Karix WhatsApp API notifications (under a 2-minute SLA), and Google Ads offline conversion tracking.
